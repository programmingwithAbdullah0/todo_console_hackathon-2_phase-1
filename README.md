# Console Todo App

A lightweight, in-memory terminal application for managing tasks.

## Features
- Create tasks with titles and descriptions
- View tasks with completion status
- Toggle completion status
- Update and delete tasks
- ISO-8601 timestamps for creation/updates

## Requirements
- Python 3.13+
- [uv](https://docs.astral.sh/uv/)

## Running the Application
```bash
uv run src/main.py
```
